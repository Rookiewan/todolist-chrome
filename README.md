# todolist-chrome
a todolist write for chrome, use indexedDB, work locally



### How to use it

```javascript
git clone https://github.com/Rookiewan/todolist-chrome.git
open chrome "chrome://extensions/"
click "加载已解压的扩展程序" and select todolist-chrome folder
```
